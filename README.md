# Mercs Royale: The Chaotic Mercenary Royale Begins!
A chaotic Team-based/Free-for-All & PvE Co-op community server project set in Team Fortress Universe, hosted from Steam game Team Fortress 2 Classified, Open Fortress and Lambda Fortress, expanding the original mode into something chaotic you never experienced.

Refer to [Wiki page](https://github.com/MercsRoyale/MercsRoyale/wiki) for details. For source codes of stuffs we have used, head to our [Codeberg](https://codeberg.org/MercsRoyale) page.
